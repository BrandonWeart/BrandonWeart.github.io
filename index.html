Research

My research focuses on applying machine learning to atmospheric science, particularly climate emulation and downscaling. I’m interested in methods that are both accurate and statistically faithful (e.g., preserving spectra and distribution tails), so they can be trusted for extremes and risk-relevant analysis.

Research themes

  • Climate emulation: ML approaches to replace computationally inefficient downscaling.
  • Generative downscaling: score-based diffusion and transfer learning across datasets.
  • Convective hazards: convective mode classification (linear/cellular/mixed) using SVRIMG.
  • Extended-range forecasting: CPC analogs and Practically Perfect Hindcasts (PPH).
  • Radar QPE + ML: Z–R relationships, regression models, and validation against ground truth sensors.

Highlights from experience

  • Created 10,000+ QC’d examples for CNN training and achieved 80%+ accuracy for storm-mode identification.
  • Worked on Chicago-area precipitation observation gaps using rain-gauge micronet concepts and radar–gauge comparisons.
  • Assisted lake-breeze detection work using computer vision and HRRR output (10-m convergence parameter).
  • Ran WRF in a pseudo-global-warming configuration for a Texas severe/flash-flood event case study.
